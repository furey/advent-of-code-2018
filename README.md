# advent-of-code-2018
Advent of Code 2018 Node.js/PHP solutions.

## Node.js

To run Node.js solutions:

```bash
node [path to solution]
```

e.g.:

```bash
node nodejs/day-01/part-1.js
```

Requirements: 

* node >= v11.2.0

## PHP

To run PHP solutions:

```bash
php [path to solution]
```

e.g.:

```bash
php php/day-01/part-1.php
```

Requirements: 

* php >= 7.2.12
